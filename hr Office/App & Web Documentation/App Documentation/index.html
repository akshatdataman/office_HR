How to change the package name?

You can change it easily from Visual Studio Code. Here are the steps:

  • In the Project pane, Find app.json file.and Open it
  • Update your Android Package Name and googleMaps under android section.
  • Update your IOS bundleIdentifier and googleMaps under ios section.

How to change app name?

  • In the Project pane, Find app.json file.and Open it
  • Enter your app name and slug inside "name" "slug"
  • Update version code.

How to change 'Splash Screen, Application Icon and MoreApp link'?

How To Change Application Icon

  • Open Android Studio > app > res >
  • Rename your icon with ic_launcher.png and paste in all mipmap folders

How To Change Splash Screen
  • Open Android Studio > app > res > drawable-nodpi > splash.png
  • Rename your splash screen image with splash.png and paste in drawable folder


How to Generate Signed APK?

Open your Terminal

  • Click Terminal > New Terminal

  • Type expo build:android and press enter
  • It will ask to create an expo account or select existing account then press enter
  • Enter UserName and press enter
  • Enter key password and press enter it will not show character on typing passsword
  • It will success login and will start building apk
  • Fill rest of the details and click OK
